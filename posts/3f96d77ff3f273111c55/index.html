gitで特定ファイルだけコンフリクト無効で強制マージ

言語ファイルなどが当てはまると思うが以下のような条件にあてはまるものを対応させたい

  • よくコンフリクトする
  • 追記のみ
  • どちらも適用させる

特定のファイルのみ

$ vim .gitattributes

*.json merge=union

すべてのファイル

$ vim .gitattributes
merge=verify

以下コマンドうったときの参考

{
    "ja": {
        "message": {
            "welcome": "ようこそ！",
<<<<<<< HEAD
            "thank": "ありがとう！"
=======
            "byebye": "さようなら！"
>>>>>>> feature/add-byebye
        }
    }
}
